Installation Notes

# Limine setup
https://wiki.archlinux.org/title/Limine

# Auto-mounting drives
If the machine has any auto-mounted drives, make sure `/etc/fstab` is set up correctly
Give user access to auto-mounted drives
`chmod user /mnt/$FOLDER_NAME`

# Package manager setup
Modify `pacman.conf` to enable the arch linux repositories
```
[system]
Include = /etc/pacman.d/mirrorlist

[world]
Include = /etc/pacman.d/mirrorlist

[galaxy]
Include = /etc/pacman.d/mirrorlist

[lib32]
Include = /etc/pacman.d/mirrorlist

# Arch repositories
[extra]
Include = /etc/pacman.d/mirrorlist-arch

[multilib]
Include = /etc/pacman.d/mirrorlist-arch
```

# Dotfiles
Clone and apply dotfiles via chezmoi
`chezmoi init https://github.com/codazendev/dotfiles.git`
`chezmoi diff`
`chezmoi apply`
(make sure to update the origin url once ssh is set up)

# Godot
Install gdvm
`curl -sSL https://gdvm.io/install.sh | bash`

# Gaming
If the machine will be used for gaming, set up scopebuddy`
`sudo curl -Lo /usr/local/bin/scopebuddy https://raw.githubusercontent.com/HikariKnight/ScopeBuddy/refs/heads/main/bin/scopebuddy`
`sudo chmod +x /usr/local/bin/scopebuddy`
`sudo ln -s scopebuddy /usr/local/bin/scb`

# osu!
If the machine will be used for osu!, blacklist the wacom and hid_uclogic kernel modules
https://unix.stackexchange.com/a/797099
https://wiki.archlinux.org/title/Kernel_module#blacklist
created file `/etc/udev/rules.d/72-hidraw-access.rules` containing:
```
KERNEL=="hidraw*", SUBSYSTEM=="hidraw*", MODE="0660", TAG+="uaccess"
```

# Firefox

`https://raw.githubusercontent.com/pixeltris/TwitchAdSolutions/f8f86706daf90daa534b26bce5b2f01238667d5f/vaft/vaft-ublock-origin.js`
in `userResourcesLocation` field in the "Advanced User Settings" screen
